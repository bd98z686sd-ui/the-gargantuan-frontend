# The Gargantuan — Frontend (Hash Routing)

Guardian-style front page + Admin at **/#/admin** (hash router, no server rewrites needed).

## Env
- `VITE_API_BASE` — your backend origin, e.g. `https://the-gargantuan-backend.onrender.com`

## Dev
```
npm i
npm run dev
```

## Build
```
npm run build
```
